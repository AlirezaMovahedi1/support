# X2Ray AI Multimodal Support Bot 🤖💬

ربات پشتیبانی هوشمند و چندرسانه‌ای **X2Ray VPN** مبتنی بر مدل **Gemini 2.5 Flash** و سامانه RAG (Retrieval-Augmented Generation).

این پروژه قادر است به صورت خودکار پیام‌های متنی مشتریان و همچنین **اسکرین‌شات‌های ارور** (از برنامه‌های v2rayNG، V2Box و...) را تحلیل کرده و پاسخ‌های دقیق و گام به گام به زبان فارسی ارائه دهد.

---

## 🌟 ویژگی‌های اصلی پروژه

1. **تحلیل متنی و تصویری (Multimodal RAG):**
   - تشخیص خودکار خطاهای تصویری در اسکرین‌شات‌ها (عدم انتخاب سرور، انقضای حجم، خاموش بودن سوئیچ اتصال).
   - تحلیل متنی هوشمند با انطباق عبارت‌های فارسی و ریشه‌یابی کلمات (مانند تطبیق «آیفونه» با «آیفون»).

2. **پایگاه دانش ساختاریافته (OKF Knowledge Base):**
   - تفکیک مستندات بر اساس موضوعات (تعرفه‌ها، واریز کارت به کارت، تنظیمات اندروید، آیفون، ویندوز و قوانین عمومی).
   - بازیابی پویا (Dynamic Retrieval) بر اساس کلمات کلیدی پیام مشتری جهت کاهش هزینه و افزایش سرعت.

3. **پردازش داده‌های تاریخی چت:**
   - استخراج بیش از ۶,۸۰۰ جفت سوال و پاسخ از ۱۰,۰۰۰ چت واقعی پشتیبانی.
   - پاکسازی داده‌ها و حذف خودکار اطلاعات حساس (شماره کارت، شماره تلفن و...).

## 📁 ساختار پروژه

```text
support/
├── bot.py                        # موتور اصلی ربات RAG و متصل به Gemini API

│   ├── android_setup.md          # راهنمای اتصال v2rayNG
│   ├── ios_setup.md              # راهنمای اتصال V2Box / Happ / Shadowrocket
│   ├── windows_setup.md          # راهنمای اتصال v2rayN / Nekoray
│   ├── pricing.md                # لیست قیمت و پکیج‌ها
│   ├── payment.md                # شماره کارت و روش‌های پرداخت
│   ├── troubleshooting.md        # راهنمای رفع اشکال متنی
│   ├── troubleshooting_visual.md # راهنمای رفع اشکال از روی اسکرین‌شات
│   └── general_rules.md          # قوانین عمومی و ساعات پاسخگویی
└── qa_dataset.json               # دیتابیس استخراج‌شده چت‌ها (فیلتر شده در gitignore)
```

## ⚙️ راه اندازی و اجرا

1. **نصب نیازمندی‌ها:**
   ```bash
   pip install google-generativeai python-dotenv pillow
   ```

2. **تنظیم کلید API:**
   یک فایل `.env` در ریشه پروژه ایجاد کرده و کلید Gemini API خود را وارد کنید:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

3. **اجرای ربات:**
   ```bash
   python bot.py
   ```

4. **اجرای تست متنی و تصویری:**
   ```bash
   python test_bot.py
   ```
